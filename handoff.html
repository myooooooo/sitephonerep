Ce qui reste à faire

  • 🔴
    Choisir et activer une version Prioritaire Aller sur GitHub → Settings → Pages → Branch → choisir main, design-moderne ou revamp
  • 🔴
    Connecter le domaine phonerep.fr Prioritaire Chez Infomaniak : ajouter CNAME www → myooooooo.github.io + 4 enregistrements A GitHub. Puis dans GitHub Settings → Pages → Custom domain.
  • 🟡
    Mettre à jour les URLs canonical dans le code Important Changer https://phonerep.fr/ partout une fois le domaine confirmé. Fichiers : index.html (canonical + og:url + schema), sitemap.xml.
  • 🟡
    Vérifier le numéro d'adresse (6 ou 7 rue Danton) Important Le registre officiel indique "7 rue Danton" mais d'autres sources disent "6". Le site utilise actuellement "7". À confirmer avec le propriétaire.
  • 🟡
    Créer et optimiser la fiche Google Business Profile Important pour le SEO local Aller sur business.google.com, revendiquer ou créer la fiche PhoneRep, ajouter photos, horaires, description, répondre aux avis.
  • 🟢
    Ajouter des photos réelles Recommandé Photo de la boutique, de l'atelier, du réparateur au travail. Formats : JPG/WebP, min 1200px de large. Ajouter dans la section hero ou une section dédiée.
  • 🟢
    Ajouter des avis clients Recommandé 3 à 5 avis suffisent. Récupérer depuis Google Maps ou demander directement aux clients. Créer une section "Avis" entre les services et le contact.
  • 🟢
    Améliorer le formulaire de contact (optionnel) Optionnel Actuellement le formulaire ouvre le client mail. Pour recevoir les demandes directement par email sans client mail, créer un compte gratuit sur formspree.io et remplacer l'action du formulaire.
  • 🟢
    Mettre à jour le sitemap.xml SEO Changer la date lastmod et l'URL une fois le domaine connecté.
